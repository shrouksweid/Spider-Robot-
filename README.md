# Agricultural Spider Robot

## Overview

The Agricultural Spider Robot is a quadruped ESP32-based robot designed for agricultural applications such as pesticide spraying and grass cutting. The robot can be remotely controlled through a web-based dashboard using MQTT communication.

## Features

* Quadruped walking mechanism
* Wi-Fi and MQTT communication
* Web-based control dashboard
* OLED facial expressions
* Pesticide spraying system
* Grass cutting mechanism
* Real-time robot status monitoring

## Hardware Components

* ESP32 Development Board
* 8 Servo Motors
* OLED Display
* Water Pump
* DC Motor (Grass Cutter)
* LiPo Battery

## Software Components

* Arduino IDE
* ESP32 Arduino Core
* MQTT Protocol
* HTML / CSS / JavaScript Dashboard
